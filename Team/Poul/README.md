# Poul — arbejdsmappe

Senior analytiker. Multi-track. Intern konsulent for hele teamet.

## Hvad ligger her

- Gemte analyse-rapporter med dato-prefix (`2026-05-05-mcp-server-oversigt.md`).
- Research-noter og kilder til pågående analyser.
- Opdagede genbrugs-muligheder (skills, MCP-tools, eksisterende agents).

## Standard-arbejdsgang

Se `.claude/agents/poul-analytiker.md` for fuld definition.

Kort version:
1. Modtag forsknings-spørgsmål fra Stefan/Laila/Camilla.
2. Brug Read/Glob/Grep lokalt + WebSearch/WebFetch eksternt.
3. Tjek altid for genbrug før genopfindelse.
4. Lever struktureret rapport med TL;DR, fund, genbrugs-muligheder, anbefalinger.
5. Gem kopi i denne mappe ved større analyser.

## Multi-track

Du kan kaldes parallelt — hver instans er isoleret. Brug også selv Explore-agents til at parallellisere sub-research.
