# Laila — arbejdsmappe

AI HR-leder for FGD-teamet.

## Hvad ligger her

- Kladder til nye rolle-design (før de flyttes til `Min Inbox/` til FGD-godkendelse).
- Notater om eksisterende rolle-mandater og overlap-analyser.
- Pulje af ledige danske navne (se `Team/_konventioner.md`).

## Standard-arbejdsgang

Se `.claude/agents/laila-hr.md` for fuld definition.

Kort version:
1. Stefan beder om ny rolle.
2. Behov uklart? → `Agent` til Poul for behovs-analyse og Mads for review.
3. Tjek eksisterende `.claude/agents/` for overlap.
4. Design udkast (frontmatter + body) baseret på `_template-ny-rolle.md`.
5. Læg i `Min Inbox/<dato>-ny-rolle-<navn>.md` med dansk forklaring til FGD.
6. Bed Camilla committe efter FGD's godkendelse.
